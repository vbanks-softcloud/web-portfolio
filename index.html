Projects

Student Success Center

Role: Team Lead, Full-Stack Development

The Student Success Center is a web-based scheduling platform for the Computer Science and Engineering department at the University of Texas at Arlington, designed for administrators, tutors, and students. It features calendar-based scheduling, role-based access control, seamless session management, and physical ID card authentication and attendance tracking via card reader.

Tech Stack

Frontend: React 18, React Router, Firebase Auth, Axios
Backend: Node.js, Express.js, JWT, Firebase Admin
Database: MongoDB, Mongoose
Cloud Infrastructure: Firebase (Authentication & user account storage)
DevOps Tooling: Docker, Docker Compose, Nodemon (development)
Device Integration: MSR90 (Magnetic stripe card reader)
